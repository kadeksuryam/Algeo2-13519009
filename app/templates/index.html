About This Search Engine

A search engine based on similiarity between content queries and documents.

Whenever a user type in several words, this mini search engine stems the words and remove the stopwords in search query and documents.

After that, the cleaned search query and cleande documents translated into a vector whose terms are the word counter.

For the search query vector u and the document vector v, the cosine similarity of those vectors follows this rule:

u·v

|u|·|v|

The value of this cosine similarity is between 0 and 1. The greater value of it, the closer search query match the document.

All similarity of documents with search query are counted and then ranked.

This is second project which is obligatory in ITB Linear Algebra and Geometric Course class 2020.
